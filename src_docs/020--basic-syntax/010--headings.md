# Headings

Markdown supports six levels of headings:

# Heading Level 1
## Heading Level 2
### Heading Level 3
#### Heading Level 4
##### Heading Level 5
###### Heading Level 6

## Alternative Syntax

You can also use underline syntax for H1 and H2:

Heading Level 1
===============

Heading Level 2
---------------

## Best Practices

- Use H1 for page titles
- Use H2 for main sections
- Use H3-H6 for subsections
- Keep hierarchy consistent
